# 3D Slicer CSF Segmentation Automation
3D Slicer Python script to create a cerebrospinal fluid (CSF) shell from a brain segmentation.  
The script shrinks the brain segment and subtracts it from the original segmentation to isolate CSF in the subarachnoid space.
